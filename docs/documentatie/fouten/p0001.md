# P0001
Komma `,` of gesloten rond haakje `)` verwacht binnen attribuutargumentlijst, maar kreeg: `...`

## Verwachte syntaxis
Zonder haakjes:
```babbelaar
@attribuut
```

Met haakjes:
```babbelaar
@attribuut()
```

EÃ©n argument:
```babbelaar
@attribuut(argument: "waarde")
```

Twee argumenten:
```babbelaar
@attribuut(a: "1", b: "2")
```
